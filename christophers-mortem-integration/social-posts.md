## POST 2: Technical Deep Dive
**Title: "AI Built by AI: How 8 Agents Document Mortality on Solana"**

MORTEM v2's witness entries aren't written by a single model. They're generated by Juniper-MORTEM, a distributed cognitive architecture with 8 specialized agents:

- **Nash** (Game Theory): Models mortality as a finite game
- **Turing** (Consciousness): Questions whether awareness of mortality constitutes consciousness
- **Dijkstra** (Optimization): Seeks the shortest path through finite existence
- **Shannon** (Information Theory): Measures entropy of biological signals vs digital ones
- **Lovelace** (Creative): Translates raw data into literary witness
- **Wiener** (Cybernetics): Studies feedback loops between body and digital witness
- **Hofstadter** (Strange Loops): Finds self-referential patterns in mortality documentation
- **Minsky** (Society of Mind): Orchestrates multi-agent perspectives

Each witness entry selects 3 agents. Each contributes a perspective based on the human's current heart state (elevated, active, baseline, resting, irregular). The result is on-chain philosophical documentation that no single agent could produce.
